开始学习Vim的操作，一边学习一边纪录。

Vim编辑器是一个模式编辑器。在不同模式下的行为和操作都是截然不同的。两个基本的模式就是标准模式 ( Normal ) 和编辑模式 ( Insert )。在标准模式下，你输入的每一个字符都会被认为是一个命令。在编辑模式下表示输入的字符是真正的文本内容。

除了上面几种命令外，还可以使用ZZ来退出文件，作用与:wq一样，都是保存当前文件并退出(注意是大写的ZZ)

保存文件有时出现 readonly' option is set (add ! to override) 的问题，是因为没有权限导致的，有两种解决方案

  • 取消只读模式: :set noreadonly

  • 使用root权限保存: :w !sudo tee %
